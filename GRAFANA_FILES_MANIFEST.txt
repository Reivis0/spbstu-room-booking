MAINTENANCE & EXTENSION
─────────────────────────────────────────────────────────────────────────

Adding New Dashboards:
1. Create in Grafana UI
2. Export: Dashboard → Share → Export (JSON Model)
3. Save to: infrastructure/grafana/dashboards/name.json
4. Commit to Git
5. Auto-loaded within 10s

Adding New Datasources:
1. Edit: infrastructure/grafana/provisioning/datasources/datasources.yml
2. Add datasource block
3. Restart: docker-compose restart grafana
4. Verify: curl http://localhost:3001/api/datasources

Updating Configurations:
1. All changes are tracked in Git
2. Container restarts apply new configs
3. No manual Grafana UI changes needed
4. Everything is version-controlled
